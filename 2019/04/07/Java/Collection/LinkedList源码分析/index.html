LinkedList同样继承自List接口，但却是基于链表实现。

LinkedList源码分析

LinkedList实现了List、Deque等接口，同时具有列表与队列的性质。LinkedList本质上是一个双向链表，在插入、删除元素上具有O(1)时间复杂度的优势，但是在获取元素时并没有ArrayList一样随机读的优势，而是要通过一个个Node进行遍历判断得到，时间复杂度为O(n)。
